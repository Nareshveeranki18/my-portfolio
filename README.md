This portfolio features a demonstration of my core full-stack capabilities through the Employee Management System (EMS).

### Overview
The EMS is a comprehensive web application designed to simplify Human Resource management for small to medium-sized companies. It allows administrators to efficiently manage personnel records, track employee details, and handle critical HR operations.

### Key Features
* **CRUD Operations:** Full functionality for Creating, Reading, Updating, and Deleting employee records.
* **Personnel Management:** Ability to add new employees, edit existing information, and manage status (transfer, promotion, termination).
* **Data Integrity:** Implemented checks to prevent duplicate entries and ensure data accuracy.
* **User Interface:** Intuitive and responsive design built using React and Bootstrap.
